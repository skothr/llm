# Sources Index

Master index of canonical sources for the LLM theory workspace.

## Papers

| Key | Title | Authors | Year | URL | Local File | Summary |
|-----|-------|---------|------|-----|------------|---------|
| vaswani2017 | Attention Is All You Need | Vaswani, Shazeer, Parmar, Uszkoreit, Jones, Gomez, Kaiser, Polosukhin | 2017 | https://arxiv.org/abs/1706.03762 | papers/vaswani2017_attention_is_all_you_need.pdf | Introduces the Transformer: encoder-decoder architecture based entirely on attention, replacing RNNs. Defines scaled dot-product attention, multi-head attention, sinusoidal positional encoding, position-wise FFN. Base model: d=512, h=8, N=6, d_ff=2048. |
| radford2018 | Improving Language Understanding by Generative Pre-Training | Radford, Narasimhan, Salimans, Sutskever | 2018 | https://cdn.openai.com/research-covers/language-unsupervised/language_understanding_paper.pdf | papers/radford2018_gpt1.pdf | GPT-1: first decoder-only transformer for language modeling. 12 layers, d=768, 12 heads, d_ff=3072. Uses learned positional embeddings, GELU activation, BPE tokenization. Introduces generative pre-training + discriminative fine-tuning paradigm. |
| brown2020 | Language Models are Few-Shot Learners | Brown, Mann, Ryder, et al. | 2020 | https://arxiv.org/abs/2005.14165 | papers/brown2020_gpt3.pdf | GPT-3: scales decoder-only transformer to 175B parameters. Demonstrates few-shot learning via in-context examples. Establishes scaling as a primary driver of capability. |
| touvron2023 | LLaMA: Open and Efficient Foundation Language Models | Touvron, Lavril, Izacard, et al. | 2023 | https://arxiv.org/abs/2302.13971 | papers/touvron2023_llama.pdf | Modern decoder-only architecture with three key modifications to original Transformer: pre-normalization with RMSNorm, SwiGLU activation (dim 2/3 * 4d), RoPE at every layer. Model sizes 7B-65B on public data. Concrete hyperparameters in Table 2. |
| sennrich2016 | Neural Machine Translation of Rare Words with Subword Units | Sennrich, Haddow, Birch | 2016 | https://arxiv.org/abs/1508.07909 | papers/sennrich2016_bpe.pdf | Introduces BPE for NLP tokenization. Algorithm: start with character vocabulary, iteratively merge most frequent adjacent pairs. Produces fixed-size vocabulary of variable-length subword units. Includes Python implementation (Algorithm 1). |
| su2021 | RoFormer: Enhanced Transformer with Rotary Position Embedding | Su, Lu, Pan, Wen, Liu | 2021 | https://arxiv.org/abs/2104.09864 | papers/su2021_rope.pdf | Rotary Position Embedding (RoPE): encodes position by rotating Q and K vectors via rotation matrix. Dot product between Q_m and K_n depends only on relative position (m-n). Properties: sequence length flexibility, decaying dependency with distance. Applied only to Q and K, not V. |
| shazeer2020 | GLU Variants Improve Transformer | Shazeer | 2020 | https://arxiv.org/abs/2002.05202 | papers/shazeer2020_glu_variants.pdf | Tests GLU variants in Transformer FFN. SwiGLU: (Swish(xW) ⊗ xV)W2. GLU variants use 3 weight matrices instead of 2; reduce hidden dim by 2/3 to match parameter count. GEGLU and SwiGLU achieve best perplexity. Adopted by LLaMA, PaLM. |
| xiong2020 | On Layer Normalization in the Transformer Architecture | Xiong, Yang, He, et al. | 2020 | https://arxiv.org/abs/2002.04745 | papers/xiong2020_layer_norm.pdf | Analyzes Post-LN (original: sublayer → add → norm) vs Pre-LN (norm → sublayer → add). Proves via mean field theory that Post-LN has gradient scale O(d√(ln d)) independent of depth, while Pre-LN scales as O(d√(ln d / L)). Pre-LN enables training without warmup. |
| zhang2019 | Root Mean Square Layer Normalization | Zhang, Sennrich | 2019 | https://arxiv.org/abs/1910.07467 | papers/zhang2019_rmsnorm.pdf | RMSNorm: simplifies LayerNorm by removing mean-centering, keeping only RMS re-scaling. Comparable performance with 7-64% faster runtime. Used by LLaMA instead of standard LayerNorm. |
